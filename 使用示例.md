# 🚀 A股微观结构监控系统 · 使用指南

本系统旨在通过自动化流程，将原始的市场碎片转化为清晰的交易逻辑。

---

## 🏃 快速执行逻辑

### 1. 全量自动化运行 (GitHub Actions 已配置)
```bash
# 这是一个三位一体的流程：
python cls_to_feishu.py           # 抓取资讯
python market_data_fetcher.py      # 抓取短线博弈数据
python market_data_consolidator.01.py  # 深度量化整合与报告生成
```

### 2. 核心产出物说明
- **每日情报 (CLS)**：财联社当日关键电报精编，按时间轴排列，标注重要资讯。
- **深度研报 (Quant)**：基于同花顺数据源的 A 股情绪量化分析 (MD 格式)。
- **数据存档 (Archive)**：最近 5 日全量市场池数据的 `.csv` 表格。

```markdown

**核心情绪定性：良好 🔴🔴🔴🔴🔴🔴🟢🟢🟢🟢**

---
### 🕰️ 因果反馈 (主流博弈)
- **昨日(非ST)表现**: 昨日涨停今日晋级: `12/35` (34.3%) | 赚钱效应: `良好`
- **题材持续性**: 持续题材: `低空经济、半导体`
- **亏钱效应监控**: 🟢 跌停 3 家 (风险可控)

### 🌡️ 市场温度计 (量能维度)

- **涨跌分布**: ⬆️ 3200 : ⬇️ 1800 (胜率: 58.0%)

### 🎯 实战核心主线 (题材共振 · 已剔除ST)
- **低空经济** (8 股涨停) | 平均换手 `12.5%`
- **人工智能** (5 股涨停) | 平均换手 `8.2%`

### 🪜 连板梯队 (身位逻辑 · 已剔除ST)
- **5板** : 某某股份(低空经济)
- **3板** : 某某科技(半导体)、某某软件(数据要素)
- **2板** : 512只略...

---
**禅师研报总结**：
> “上善若水，顺势而为，关注主线轮动。”
```

---

## 💡 实战建议

1. **看身位**：关注“连板梯队”中的最高板，它决定了市场向上开拓的空间。
2. **看晋级**：若“昨日晋级率”低于 15%，说明短线博弈恶劣，宜空仓坐禅。
3. **看题材**：重点参与“持续题材”中平均换手在 5%-15% 之间的个股，避免缩量加速或高位放巨量。

## ❓ 常见问题

**Q: 为什么搜不到某些股票的涨停信息？**
A: 本系统**自动剔除了 ST 和 *ST 个股**，旨在关注真正具有主流资金合力的标的，避免受壳资源炒作干扰。

**Q: 历史数据去哪了？**
A: `output/date/` 目录默认只展示最近 5 天的数据，更早的数据会自动移入 `history/` 文件夹中。
